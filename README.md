# Free Fire — Bootstrap 5.2 Site

A bold, dark-themed fan landing page built with **Bootstrap 5.2.3**.

## Run
Just open `index.html` in a browser. All dependencies (Bootstrap, Bootstrap Icons, Google Fonts) load via CDN.

## Structure
- `index.html` — all sections (hero, features, characters, weapons, maps, tournaments, news, CTA, footer)
- `css/styles.css` — custom theme on top of Bootstrap
- `js/main.js` — small scroll enhancement

## Customize
- Colors: edit `:root` in `css/styles.css` (`--ff-orange`, `--ff-red`, etc.)
- Fonts: swap Google Fonts link in `index.html`
- Images: drop your own into `images/` and replace gradient placeholders with `<img>` tags
